
自做usb数字麦克风, 步骤如下
1.硬件准备
   A.teensy3.6版本
   B.MSM261S4030H0 或 INMP441数字麦克风模块
   C.MAX98357A 模块 (如果不要声音输出，可不接)
2. 软件烧录至teensy 
    A.下载安装软件arduino
    B. 安装 teensy 开发库，
    C. 打开源码,选择开发板，烧录
